المميزات الرئيسية

بنية معمارية حقيقية تخلّي الـ AI يتفاعل مع نظامك بشكل آمن ومنظّم، مش مجرد توليد نصوص.

⚙

Dependency Injection

كل خدمة (config, logger, providers, memory, event bus) مسجّلة مرة واحدة في Container — استبدال أي خدمة بنسخة اختبار سهل.

⚡

Event-Driven Architecture

الأحداث القياسية (CommandExecuted, TaskStarted, PluginLoaded…) تنتشر عبر EventBus — أي جزء يقدر يستمع بدون ارتباط بالناشر.

🔧

Tool Calling حقيقي

الـ AI يستقبل أدوات حقيقية بصيغة JSON Schema (read_file, write_file, git_commit, run_shell…) وينادي عليها مباشرة، مش توليد أوامر نصية.

🧠

Context Manager ذكي

يجمع صورة شاملة قبل أي طلب: المحادثة، المجلد، حالة Git، نظام التشغيل، الملفات المفتوحة، آخر الأوامر والأخطاء.

🎭

AI Roles & Profiles

أدوار متخصصة (Coder, Reviewer, Linux Expert, Security…) مع Profiles مخصّصة حسب نوع المستخدم (Developer, Pentester, Student…).

📂

Multi-Session & Jobs

إدارة عدة جلسات في نفس الوقت — list, switch, resume, archive. مع Job Manager يشغّل أوامر في الخلفية (bg, jobs, fg, kill).

🔄

Workflow + Macro

ابني سير عمل قابل للحفظ والتصدير، وسجّل ماكروز من استخدامك الحقيقي ثم أعد تشغيلها بنقرة واحدة.

🧹

Plugin Marketplace

Marketplace عبر ملف JSON مركزي على GitHub — plug install/search/update/publish لأنظمة إدارة الإضافات كاملة.

📑

Layered Config

4 طبقات (session > workspace > project > global). شارك إعدادات الفريق عبر .cai.json بدون لمس الإعدادات الشخصية.

🛡

أمان واستقرار

Secrets Scanner، Snapshot/Undo، Error Database تراكمية، Crash Recovery، Offline Mode، Auto Update + Rollback، Terminal Recorder.

🩹

أوامر تشخيصية

doctor لفحص شامل، whoami لملخّص الحالة، metrics لإحصائيات الأداء وتقرير التكلفة.

📱

دعم Termux (Android)

تكيّف تلقائي مع Android عبر Termux — أوامر مخصّصة للبطارية، الإشعارات، GPS، TTS، والوصول للتخزين.

الأوامر حسب نظام التشغيل

دليل مختصر وسريع — لو عايز الدليل الكامل خطوة بخطوة ادخل قسم "التشغيل" تحت.

🐧 Ubuntu / Debian / Kali

# 1. تأكد من وجود Python ثم ثبّت الأساسيات sudo apt update sudo apt install -y python3 python3-pip python3-venv git # 2. فك ضغط المشروع cd ~ && unzip cai_project.zip && cd cai_project # 3. بيئة افتراضية + تثبيت python3 -m venv venv source venv/bin/activate pip install --upgrade pip pip install -r requirements.txt pip install -e . # 4. التشغيل cai # أو: python3 -m cai

💡 لإضافة اختصار دائم: echo 'alias cai-start="cd ~/cai_project && source venv/bin/activate && cai"' >> ~/.bashrc

🎩 Fedora / Arch

# Fedora: sudo dnf install -y python3 python3-pip git # Arch: sudo pacman -Sy python python-pip git # باقي الخطوات مطابقة لـ Ubuntu python3 -m venv venv source venv/bin/activate pip install --upgrade pip pip install -r requirements.txt pip install -e . cai

🍎 macOS (Homebrew)

# لو مفيش Homebrew: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" brew install python git # باقي الخطوات cd ~ && unzip cai_project.zip && cd cai_project python3 -m venv venv source venv/bin/activate pip install --upgrade pip pip install -r requirements.txt pip install -e . cai

💡 لو ظهر خطأ في cryptography: xcode-select --install

💻 Windows — PowerShell (موصى به)

# 1. نزّل Python 3.9+ من python.org (فعّل Add to PATH أثناء التثبيت) # 2. نزّل Git من git-scm.com # 3. فك ضغط المشروع Expand-Archive -Path cai_project.zip -DestinationPath . cd cai_project # 4. اسمح بتشغيل السكريبتات (مرة واحدة فقط) Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned # 5. بيئة افتراضية + تثبيت python -m venv venv .\\venv\\Scripts\\Activate.ps1 pip install --upgrade pip pip install -r requirements.txt pip install -e . # 6. التشغيل cai

💻 Windows — CMD

cd C:\\Users\\<اسمك>\\cai_project venv\\Scripts\\activate.bat pip install --upgrade pip pip install -r requirements.txt pip install -e . cai

ملاحظة: خطوة Set-ExecutionPolicy مش مطلوبة في CMD، خاصة بـ PowerShell بس.

📱 Android عبر Termux (حمّل من F-Droid فقط)

# 1. حدّث الحزم pkg update -y && pkg upgrade -y # 2. ثبّت الأساسيات pkg install -y git python # 3. انقل المشروع (لو zip في Downloads) termux-setup-storage cp ~/storage/downloads/cai_project.zip ~/ cd ~ && unzip cai_project.zip && cd cai_project # 4. شغّل سكريبت التثبيت الجاهز (موصى به) bash termux_setup.sh # أو التثبيت اليدوي pkg install -y python git clang libffi openssl rust termux-api pip install --upgrade pip pip install -r requirements.txt pip install -e . cai

📡 للأوامر الخاصة بالهاتف: phone battery · phone notify · storage · ollama connect <IP>
